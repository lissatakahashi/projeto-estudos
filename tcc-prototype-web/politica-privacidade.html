Política de Privacidade

Última atualização: 15/03/2026

Esta política descreve como o Projeto de Estudos coleta, utiliza e protege os dados pessoais dos usuários, em conformidade com a Lei Geral de Proteção de Dados Pessoais (LGPD - Lei nº 13.709/2018).

1. Quais dados coletamos

Durante o cadastro, podemos coletar os seguintes dados:

  • Nome completo
  • E-mail
  • Data de nascimento
  • Celular

2. Finalidade do uso dos dados

Os dados são utilizados para:

  • Criar e gerenciar sua conta na plataforma;
  • Permitir autenticação segura e recuperação de acesso;
  • Personalizar sua experiência e registrar progresso de estudos;
  • Viabilizar comunicações essenciais sobre o uso da conta.

3. Armazenamento e segurança dos dados

Os dados são armazenados em infraestrutura digital com controles de acesso e medidas técnicas razoáveis para proteger as informações contra acesso não autorizado, perda ou alteração indevida.

Os dados são mantidos apenas pelo tempo necessário para cumprir as finalidades desta política, salvo obrigações legais de retenção.

4. Direitos do titular segundo a LGPD

Você pode, a qualquer momento, solicitar:

  • Confirmação da existência de tratamento dos seus dados;
  • Acesso aos dados pessoais armazenados;
  • Correção de dados incompletos, inexatos ou desatualizados;
  • Anonimização, bloqueio ou eliminação de dados desnecessários;
  • Revogação do consentimento, quando aplicável.
